## 📖 About

The **Retirement Planning Guide** is a single-file HTML website that provides comprehensive, verified retirement information for Americans approaching or entering retirement. There is no backend, no database, no API key, no sign-up, and no data collection of any kind.

Everything is built into one `index.html` file and hosted completely free on GitHub Pages.

## 🚀 Deploy to GitHub Pages (5 Minutes, Free)

### Step 1 — Create a Repository

1. Go to [github.com](https://github.com) and sign in (or create a free account)
2. Click **+** → **New repository**
3. Name it `retirement-hub` (or any name you prefer)
4. Set visibility to **Public**
5. Click **Create repository**

### Step 2 — Upload the Files

1. In your new repository, click **Add file → Upload files**
2. Upload both files:
   - `index.html` ← the main site file (rename `retirement_hub.html` to `index.html`)
   - `README.md` ← this file
3. Add a commit message like `Initial upload — Retirement Planning Guide`
4. Click **Commit changes**

### Step 3 — Enable GitHub Pages

1. Go to your repository → **Settings** tab
2. Scroll down to **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Set branch to **`main`** and folder to **`/ (root)`**
5. Click **Save**

### Step 4 — Access Your Live Site

After 1–2 minutes, your site will be live at:

```
https://YOUR-USERNAME.github.io/retirement-hub
```

GitHub will show you the exact URL in **Settings → Pages** once it's deployed.

> **Tip:** Bookmark the URL and share it with family members who are planning their retirement.

## 🔄 Keeping It Updated

Government figures change annually (usually in October–November for the following year). To update:

1. Open `index.html` in any text editor
2. Search for the value you want to update (e.g. `$185` for Medicare Part B)
3. Replace with the new figure
4. Update the source citation year if needed
5. Commit and push — GitHub Pages updates automatically within minutes

**Key annual update dates:**
- **October** — Medicare Part B/D premiums announced for next year (CMS.gov)
- **October/November** — IRS contribution limits for next year (IRS.gov)
- **October** — Social Security COLA (cost of living adjustment) announced (SSA.gov)
- **January 1** — New figures take effect
